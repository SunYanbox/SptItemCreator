## 步骤1：复制模板

复制模组本体中`模组文件夹/data/模板-Template`文件夹下的模板文件（.sic.jsonc）

复制并重命名到`模组文件夹/data`及其子文件夹下的任意位置

例如:

- `模组文件夹/data/你的新物品-YourNewItem.sic.jsonc`
- `模组文件夹/data/子弹Ammo/你的新子弹-YourNewAmmo.sic.jsonc`
- `模组文件夹/data/某个作者名称AuthorName/他的药品HisMedicine/他的药品-HisMedicine.sic.jsonc`

### 步骤2：配置关键字段

```json
{
  "$type": "common",
  "enable": true,
  "baseInfo": {
    "id": "6900c8e93ea877662a000001",
    "name": "我的自定义物品",
    "parentId": "基础物品ID"
  },
  "propertyOverride":{
      
  }
}
```

1. 参考`游戏文件夹\SPT\SPT_Data\database\templates\items.json`检索数据

通过从你的`游戏文件夹\SPT\SPT_Data\database\templates\items.json`中寻找你需要的属性, 并把那些属性复制给`propertyOverride`即可

2. 参考数据结构文件夹下给出的属性解释修改提供的字段的值

### 步骤3：添加模型（可选）

1. 模型文件放入`bundles`文件夹
2. [在`bundles.json`中注册模型](添加自定义模型.md)
3. 在物品配置中引用路径
