## ammoType可用的值

> 需要像示例一样使用下方三个字符串之一

- `bullet`
- `buckshot`
- `grenade`

本属性用于弹药，具体含义如下：

- `bullet`：**子弹**
- `buckshot`：**铅弹**（霰弹枪用的弹丸）
- `grenade`：**手榴弹**

**示例**

```jsonc
{
	"_id": "54527ac44bdc2d36668b4567",
    "_name": "patron_556x45_M855A1",
    "_parent": "5485a8684bdc2da71d8b4567",
    "_type": "Item",
    "_props": {
        "ammoType": "bullet"
        // 其他属性在此次省略
    }
}
```
