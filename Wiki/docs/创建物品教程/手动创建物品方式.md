### 步骤1：复制模板

复制`data`文件夹下的模板文件（.sic.jsonc）

### 步骤2：配置关键字段

```json
{
  "$type": "common",
  "enable": true,
  "baseInfo": {
    "id": "6900c8e93ea877662a000001",
    "name": "我的自定义物品",
    "parentId": "基础物品ID"
  },
  "propertyOverride":{
      
  }
}
```

通过从你的`游戏文件夹\SPT\SPT_Data\database\templates\items.json`中寻找你需要的属性, 并把那些属性复制给`propertyOverride`即可

### 步骤3：添加模型（可选）

1. 模型文件放入`bundles`文件夹
2. [在`bundles.json`中注册模型](3. 添加自定义模型.md)
3. 在物品配置中引用路径
